# MTR4 E233系パック修正版
[このパック](https://modrinth.com/resourcepack/mtr4-jre233) について以下を修正しました。

- 故障中の表記を回送に変更しました。
- TC Back（後部先頭車）の台車参照が同作者の別作品Sotetsu 11000 系素材を指していたため、対応する E233 台車参照へ修正しました。
- 窓から生じる水の透明化、信号現示が表示されない、シェーダー適応時に窓が真っ黒になる問題を解消しました。

また、次の仕様を変更しています。
- 広告スペースを進急電鉄に変更
- LCDのROMを差し替え種別(MTR内の系統番号)を表示するように変更
